## 🚩 Caracteristicas

* users and roles handling.
* docker file for database.
* authtentication and roles segmentation.
* nodeJs

## Instalacion
Ejecutar dentro del directorio raiz del proyecto.

```bash
npm install
```
## Ejecutar proyecto
start databse.

```bash
docker-compose up -d
```
